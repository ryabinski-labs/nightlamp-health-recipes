# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**.

- Canonical text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- Translations: https://www.contributor-covenant.org/translations
- FAQ: https://www.contributor-covenant.org/faq

The full text governs participation in this project's issues, pull requests,
discussions, and any community spaces operated under this repository.

## Reporting

Report concerns to **cigan1@gmail.com**. All reports are handled
confidentially. Reports are reviewed by the maintainers listed in
[`MAINTAINERS.md`](MAINTAINERS.md).

If a report involves a maintainer, send it to another maintainer instead.
